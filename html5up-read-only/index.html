Foreign Speech Translation and Subtitles

Transcribe and translate korean videos with AI.

The goal of our project is to create a working transcription and translation of Korean videos utilizing AI. We wanted to compare the effectiveness of different translator models on the Korean language. The three models used are GoogleTranslate, Microsoft Translator, and DeepL. Google translate and Microsoft Translator are both available in the deep-translator Python library while DeepL API was used for Korean translations.
